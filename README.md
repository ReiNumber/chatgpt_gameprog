# ChatGPTで楽々ゲームを作る「はじめてのPythonプログラミング」

電子書籍「ChatGPTで楽々ゲームを作る - はじめてのPythonプログラミング」で扱ったプログラムを提供します。
対象の電子書籍はAmazon「https://www.amazon.co.jp/dp/B0CFHCG4QG」で購入できます。Kindle Unlimitedに加入している場合は、無料で閲覧可能です。
